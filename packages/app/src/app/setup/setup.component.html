Balnc Setup

Balnc (balance) is an free open source management suite for small businesses and freelancers. It has a collection of pre-build modules and the tools for software developers to create new ones.

All modules are running local at the users machine/browser. This improves loading speed to a magnitute similar of the old installable applications.

Your data may be configured to also be stored localy. It is not require a database server for storing them. As so, they are not visible to any third parties and the speed is improved even more.
